Influxcoin (INFX) WHMCS Payment Module for use with the Influx Payment Processing API 

Steps to insall WHMCS Module 

1. Unzip
2. Upload the contents to /modules/gateways in your WHMCS installation
3. Sign in to WHMCS as Admin
4. Go to Setup>Payments>Payment Gateways and click on the All Payment Gateways tab
5. Select INFXcoin and activate the module 
6. Enter the INFX address you want the hosting funds deposited to
7. Click the save changes button and your now ready to start accepting INFX payment using our processor
